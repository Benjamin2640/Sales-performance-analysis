# Sales-performance-analysis
This repo contains a beginner friendly sales analysis project done in excel
## tool used
- microsoft excel
- google sheet


## project overview
This project involve analyzing Adidas sales data to identify key performance indicators (KPIs), evaluate sales performance, and uncover trends across products, regions, sales channels, and retailers. Using Excel for data cleaning, analysis, and visualization, the project generated data-driven insights and strategic recommendations aimed at improving sales performance, profitability, inventory management, and overall business growth.


## what i learned
- How to clean messy excel data using formulas
- How to summarize KPIS using pivot tables
- How to visualize insight using excel
